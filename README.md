# New Jersey Window Experts Website

## File Labels
- `index.html`: Main website markup and page content.
- `CNAME`: GitHub Pages custom domain config.

## CSS Files (`assets/css/`)
- `base.css`: Global element defaults (body, headings, box sizing).
- `components.css`: Reusable component styles (buttons, cards, accordion, forms, headers).
- `layout.css`: Grid and layout structure rules.
- `responsive.css`: Media-query overrides for tablet/mobile breakpoints.
- `main.css`: CSS entrypoint importing all CSS section files.

## JavaScript Files (`assets/js/`)
- `navigation.js`: Page tab switching and mobile nav toggle.
- `accordion.js`: FAQ accordion open/close behavior.
- `form.js`: Contact form submit button feedback behavior.
- `main.js`: Startup initialization (`showPage('home')` on load).

## Scripts (`scripts/`)
- `local-static-server.mjs`: Local HTTP static file server for previewing the site.

## Documentation (`docs/`)
- `frontend-implementation-rules.md`: Internal workflow and design rules for frontend work.
